# Kinde Custom CSS Properties Starter

A lightweight starter focused on adding custom CSS properties for Kinde's Custom UI.

## Customization Guide

### CSS Properties

Update the CSS variables returned from `kindeSrc/styles/styles.ts` to change colors, spacing, and other visual tokens used by Kinde's UI.
